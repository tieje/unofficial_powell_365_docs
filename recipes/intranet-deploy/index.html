    In Powell Manager 

    Create a Navigation 

    1. Go to Intranet Deploy Navigation
    2. Create a new navigation
    3. Fill in fields
      1. Title
      2. Check Top Navigation
      3. Navigation Type, choose Custom navigation (cross site collections)
    4. Create new nodes

    Add to Your Site 

    1. Go to Site Collections
    2. Choose your site
    3. Configuration (left column)
    4. Site Collection Features, Top Navigation, choose the navigation you created.
