    Unofficial Powell 365 Documentation 

      Powell 365, affectionately called "Paolo", is a site manager for SharePoint.

    Motivation 

      The purpose of this documentation is to ease the developer experience of Powell 365. As of this writing (2023/01/19), official Powell documentation could be a bit more specific. While official training videos do exist:

    • building a theme
    • site collection deployment
    • Powell apps

    as well as a learning center (email Powell for access), this form of training is not especially helpful for developers building SharePoint sites using Powell.
