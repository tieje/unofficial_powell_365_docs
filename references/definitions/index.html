    Purpose 

      The purpose of this page is to provide definitions to commonly used terms throughout Powell.

    Terms 

    Portal 

      The Portal site collection URL will let you to retrieve different features as the Portal application configure, My links, the notification and more. For those feature we will retrieve the configuration in hte site coll defined as the portal site collection url.

      For example, if the user has notifications, these notifications will follow the user across site collections from the specified portal site collection URL.

    Save 

      A button that saves the configuration to Powell Manager. The more shallow the button's site map location, the more it saves.

    Sync 

      A button that creates a deployment task to the tenant. The more shallow the button's site map location, the more items it tries to sync, which may result in significantly longer sync wait times.
